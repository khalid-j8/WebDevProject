# Apex Motors

A luxury car dealership web application built with PHP and MySQL.

**Live Site:** [webdevproject-production-aade.up.railway.app](https://webdevproject-production-aade.up.railway.app)

**Repository:** [github.com/khalid-j8/WebDevProject](https://github.com/khalid-j8/WebDevProject)

---

## Requirements

- PHP 8.0 or higher
- MySQL 8.0 or higher

## Installation

1. Clone the repository:
```bash
git clone https://github.com/khalid-j8/WebDevProject.git
cd WebDevProject
```

2. Create the database and import the schema:
```bash
mysql -u root -p < database/schema.sql
```

3. Start the server:
```bash
php -S localhost:8000
```

4. Open [http://localhost:8000](http://localhost:8000) in your browser.
